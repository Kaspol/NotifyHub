## Минимальный MVP:

- **Auth**: регистрация, логин, refresh token.
- **Организации и роли**: owner, manager, viewer.
- **Кампании**: создать, запланировать, запустить тест.
- **Templates**: хранение message body и variables.
- **Webhooks**: endpoint URL + signing secret.
- **API keys**: выпуск и отзыв ключей.
- **Audit log**: кто что создал, удалил, запустил.
